Real Estate - Price Prediction

Real Estate Price Analysis and Prediction – Greater Tunis Area

This project analyzes the real estate market in the Greater Tunis area and predicts property prices using machine learning models. Data was collected through web scraping and cleaned for analysis using Python libraries. Various regression algorithms were tested to ensure high prediction accuracy. The final solution was deployed with Streamlit for an interactive and user-friendly experience..

Project Context

The Tunisian real estate market, especially in the Greater Tunis region, is dynamic yet lacks structured data insights. This project aimed to:

• Collect local real estate data from multiple online sources.

• Analyze market trends and influential factors.

• Build predictive models for property prices.

• Provide an interactive web application for users to estimate property values easily.

Methodology

The project followed the CRISP-DM (Cross Industry Standard Process for Data Mining) methodology

• Business Understanding: Identify key factors influencing property prices (location, size, number of rooms, etc.).

• Data Understanding: Gather data from various real estate listing websites through web scraping.

• Data Preparation: Clean, preprocess, and normalize the data to handle missing or inconsistent values.

• Modeling: Train and evaluate multiple regression models.

• Evaluation: Select the best model based on accuracy and error metrics.

• Deployment: Develop an interactive web app with Streamlit for end-users.

Modeling and Performance

Three regression algorithms were implemented and compared:

• Linear Regression

• Random Forest Regressor

• Gradient Boosting Regressor

The Gradient Boosting model demonstrated superior performance with a MAPE (Mean Absolute Percentage Error) below 10%, delivering reliable price predictions especially in densely populated urban areas such as Tunis, Ariana, and La Marsa.

Interactive Web Application with Streamlit

To enhance accessibility, an intuitive web app was developed using Streamlit. It allows users to:

• Visualize regional price trends

• Input property features

• Receive instant price estimates

making the predictive insights accessible to real estate buyers, investors, and agencies.

Results and Impact

• Clear visualizations and comprehensive analysis of regional real estate trends.

• A robust predictive model with high accuracy for price estimation

• An interactive platform that can be scaled and adapted to other regions or markets

Key Learnings

• End-to-end project management using a structured data science process (CRISP-DM)

• Handling complex web scraping and data preprocessing tasks.

• Comparing and tuning machine learning regression models for real-world data.

• Deploying a data product with Streamlit for a seamless user experience.
